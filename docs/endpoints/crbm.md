# CRBM - Conselho Regional de Biomedicina (Em Beta)

!!! warning "Em Beta"

    Essa API está em Beta, entre em contato com o [Suporte](../suporte.md) para solicitar acesso.

## Introdução

Esta API permite consultar e buscar informações sobre profissionais e
estabelecimentos registrados nos Conselhos Regionais de Biomedicina (CRBM) do Brasil.
