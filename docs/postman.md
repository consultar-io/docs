# Coleções

Temos coleções para as APIs do Consultar.IO em diferentes formatos para facilitar o desenvolvimento e teste das APIs.

- [Postman](#postman)
- [Insomnia](#insomnia)
- [Bruno (OpenCollection)](#bruno-opencollection)

## Postman

[<img src="https://run.pstmn.io/button.svg" alt="Run In Postman" style="width: 128px; height: 32px;">](https://god.gw.postman.com/run-collection/49657121-67b8bdd6-d2a3-4670-919d-23be3058fa49?action=collection%2Ffork&source=rip_markdown&collection-url=entityId%3D49657121-67b8bdd6-d2a3-4670-919d-23be3058fa49%26entityType%3Dcollection%26workspaceId%3Daff38029-3b6a-4292-a751-b410e14cec19)

Clique no botão ou acesse a coleção no Postman: [https://www.postman.com/consultar-io/apis-consultar-io/](https://www.postman.com/consultar-io/apis-consultar-io/overview)

Você também pode fazer o download do [APIs.postman_collection.json](https://github.com/consultar-io/postman/blob/main/APIs.postman_collection.json) e importar a coleção.
