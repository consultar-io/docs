# Coleções

Temos coleções para as APIs do Consultar.IO em diferentes formatos para facilitar o desenvolvimento e teste das APIs.

- [Postman](#postman)
- [Insomnia](#insomnia)
- [Bruno (OpenCollection)](#bruno-opencollection)
- [OpenAPI](#openapi)

## Postman

[<img src="https://run.pstmn.io/button.svg" alt="Run In Postman" style="width: 128px; height: 32px;" width="128" height="32">](https://god.gw.postman.com/run-collection/49657121-67b8bdd6-d2a3-4670-919d-23be3058fa49?action=collection%2Ffork&source=rip_markdown&collection-url=entityId%3D49657121-67b8bdd6-d2a3-4670-919d-23be3058fa49%26entityType%3Dcollection%26workspaceId%3Daff38029-3b6a-4292-a751-b410e14cec19)

Clique no botão ou acesse a coleção no Postman: [https://www.postman.com/consultar-io/apis-consultar-io/](https://www.postman.com/consultar-io/apis-consultar-io/overview)

Você também pode fazer o download do [APIs.postman_collection.json](https://github.com/consultar-io/postman/blob/main/APIs.postman_collection.json) e importe a coleção.

## Insomnia

[<img src="https://insomnia.rest/images/run.svg" alt="Run in Insomnia" style="width: 128px; height: 32px;" width="128" height="32">](https://insomnia.rest/run/?label=APIs%20Consultar.IO&uri=https%3A%2F%2Fraw.githubusercontent.com%2Fconsultar-io%2Finsomnia%2Frefs%2Fheads%2Fmain%2Finsomnia.yaml)

Clique no botão ou faça o download do [insomnia.yaml](https://github.com/consultar-io/insomnia/blob/main/insomnia.yaml) e importe a coleção no [Insomnia](https://insomnia.rest/).

## Bruno

[<img src="https://fetch.usebruno.com/button.svg" alt="Fetch in Bruno" style="width: 128px; height: 32px;" width="128" height="32">](https://fetch.usebruno.com?url=https%3A%2F%2Fgithub.com%2Fconsultar-io%2Fbruno.git)

Clique no botão ou faça o clone do repositório [https://github.com/consultar-io/opencollection](https://github.com/consultar-io/opencollection) e abra o repositório no [Bruno](https://www.usebruno.com/).
